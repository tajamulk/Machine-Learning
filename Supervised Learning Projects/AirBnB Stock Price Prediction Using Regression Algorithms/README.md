# AirBnB Stock Price Prediction Using Regression Algorithms

###

Objective of this project is to predict AirBnB Stock Prices.

## Tech Stack

**Language:** Python

**Libraries:** numpy, pandas, seaborn, matplotlib, scipy, sklearn.

## Appendix

* Exploratory Data Analysis
* Models Used: 
    * LinearRegressor, 
    * LogisticRegressor
